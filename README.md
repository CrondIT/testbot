# Многофункциональный Telegram-бот с ИИ

## Описание
Этот бот предоставляет доступ к различным ИИ-моделям (OpenAI, Google Gemini, DALL-E) через Telegram интерфейс. Поддерживает чат, поиск в интернете, генерацию и редактирование изображений, а также систему монетизации через Telegram Stars.

## Возможности
- Чат с ИИ (OpenAI GPT-4o-mini)
- Поиск в интернете с помощью ИИ
- Генерация изображений (DALL-E 3)
- Редактирование изображений (Google Gemini)
- Система монетизации через Telegram Stars

## Ограничения по токенам
Бот включает в себя систему ограничения размера токенов для используемых моделей:

- **OpenAI GPT-4o-mini**: Максимум 128,000 токенов, с резервированием 1,000 токенов для ответа
- **DALL-E 3**: Максимум 1,000 токенов для промптов
- **Google Gemini 2.5 Flash**: Максимум 1,048,576 токенов
- История сообщений ограничена 10 последними сообщениями
- Автоматическое сокращение контекста при превышении лимита

## Команды
- `/start` - Начать работу с ботом
- `/ai` - Режим обычного чата
- `/ai_internet` - Режим чата с поиском в интернете
- `/ai_image` - Режим генерации изображений
- `/ai_edit` - Режим редактирования изображений
- `/billing` - Управление счетом
- `/clear` - Очистить контекст текущего режима

## Установка
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте переменные окружения в файле `.env`
3. Запустите бота: `python bot.py`

## Требования
- Python 3.8+
- Библиотеки из requirements.txt
- API-ключи от OpenAI, Google Gemini и Telegram
